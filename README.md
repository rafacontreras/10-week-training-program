# Training Program Tracker · Version 2.4

A mobile-first 10-week glute-focused training tracker for iPhone Safari or GitHub Pages.

## Version 2.4 updates
- Weekly planner now lets the user choose each day as **Workout** or **Recovery**.
- Removed the older move/swap dropdown system.
- Each week stays capped at 4 strength days so the program remains structured.
- Calendar tab again shows each week’s purpose, such as Foundation, Build, Shape, Strength Endurance, and Peak & Polish.
- Keeps progress rings, next uncompleted workout, motivational quotes with attribution, skip for recovery days, confetti, export/import backup, and offline PWA support.

## Use locally
Open `index.html` in a browser. Progress saves to that browser using localStorage.

## Publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files in this folder.
3. Go to Settings > Pages.
4. Choose the main branch and root folder.
5. Open the published GitHub Pages URL on iPhone Safari.
6. Tap Share > Add to Home Screen.

## Note
Progress is stored locally on the device/browser. Clearing Safari website data will erase the tracker unless a backup was exported.
